# Hotpot Golang SDK

A Golang SDK for [Hotpot](https://docs.hotpot.tech/), a cross-chain DEX aggregator that enables token swaps across Ethereum, Solana, Tron, and Bitcoin.

## Installation

Add this dependency to your project:

```bash
go get github.com/BoostyLabs/hotpot-sdk-go@latest
```
